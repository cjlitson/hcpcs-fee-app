# VA HCPCS Fee Schedule Manager — v1.1.1

**Release date:** 2026-03-17  
**Download:** `HCPCSFeeApp.exe` (Windows standalone)

This is a patch release focused on documentation and packaging stability.

---

## Fixed / Improved

- **Documentation** — README rewritten with a prominent Quick Start (Windows EXE) section, a Troubleshooting section (SmartScreen/unsigned EXE, ODBC/SQL Publisher prerequisites, CMS sync failures and manual import fallback), and improved formatting throughout.
- **Packaging / stability** — Minor build and packaging refinements for the standalone Windows executable (`HCPCSFeeApp.exe`).

---

## Installation / Upgrade

1. Download `HCPCSFeeApp.exe` from [GitHub Releases](https://github.com/cjlitson/hcpcs-fee-app/releases/tag/v1.1.1).
2. Replace your existing copy (if any).
3. Run the EXE — no installer required.

---

## System Requirements

- **Windows 10/11** (for the packaged `.exe`)
- **Python 3.11+** (only if running from source)
