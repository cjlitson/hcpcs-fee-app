VA HCPCS Fee Schedule Manager — Installation
=============================================

1. Extract this ZIP file to any temporary location
2. Double-click "Install.bat" at the root of the extracted folder to run the installer
3. The app will be installed to your Documents folder
4. A desktop shortcut will be created automatically
5. The app will launch automatically when installation completes

Note: The "AppFiles" folder inside the ZIP contains the application files.
You do not need to open it — just double-click Install.bat.

To run the app later, double-click the desktop shortcut or navigate to:
  Documents\HCPCSFeeApp\HCPCSFeeApp.exe

No administrator rights are required.
PowerShell is NOT required.
